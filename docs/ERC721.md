# ERC721 Token

`constructor`

``` js
    constructor(
        string memory _name, 
        string memory _symbol
    )
```

### Data-type 

| VARIALBLE          | TYPE        | EXAMPLE       |
| ------------------ | ----------- | ------------- |
| __name_            | string      | "Basic NFT"   |
| __symbol_          | string      | "BNFT"        |
## ERC721 API

`name`  
``` js
    name() -> string
    // method: callcontract  
```
`symbol`  
``` js
    symbol() -> string
    // method: callcontract  
```

`totalSupply`  
``` js
    totalSupply() -> uint256
    // method: callcontract  
```

`balanceOf`  
``` js
    balanceOf(address _account) -> uint256
    // method: callcontract  
```

`transfer`
``` js
    transfer(address _account, uint256 _amount) -> bool
    // method: sendtocontract  
```
